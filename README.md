# EatDaTaco

Create custom tacos, eat tacos, and view tacos (both eaten and yet-to-be-eaten)!

To use our web app, simply start adding tacos and eating as you please. Devoured tacos will be displayed to the right.

## Technologies Used
- JavaScript
- Node
- Express
- Handlebars
- MVC
